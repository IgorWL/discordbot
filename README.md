# discordbot

In reality this repository is from a bot called; ReverseBot, he is a bot focused on many cool things


# How To Use
Para usar é simples!


no ``config.json`` coloque isso:
```
{
"prefix": "bot prefix"
}
```

Após isso crie um arquivo chamado: ``.env``:
```
TOKEN= Token do Bot
```
